# EmployeeApps

EmployeeApps is a simple Employee Management Web Application developed using Java Servlet technology.

## 🚀 Technologies Used

- Java
- JDBC
- Servlet
- HTML
- MySQL Database
- Apache Tomcat Server
- Eclipse IDE
- Git & GitHub

## 📂 Features

- Add Employee Details
- Store Employee Data in MySQL Database
- Dynamic Web Application
- Database Connectivity using JDBC
- Simple and User-Friendly Interface

## 🛠 Tools & Software

- Eclipse IDE for Enterprise Java Developers
- Apache Tomcat v10
- MySQL
- Git Bash
- GitHub

## 📁 Project Structure

- `src/main/java` → Java Servlet Files
- `src/main/webapp` → HTML & Web Files
- `WEB-INF` → Configuration Files
- `web.xml` → Deployment Descriptor

## 🔗 Database

MySQL database is used to store employee records.
